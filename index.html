Installation Instructions

Learn how to download, install, and play this game with step-by-step instructions!

1st step: Download the game!

2nd step: Extract the game using WinRAR or 7-Zip, and open the executable from the extraction location.


3rd step: Allow Windows to open the game setup assistant by clicking "More Info" and then "Allow".

4th step: And again, by clicking "Yes".


5th step: In the setup assistant, ensure you have enough free disk space and choose an installation path, and click "Next".
Check the box if you want to create a desktop shortcut for the game and click "Next" again to proceed to the confirmation screen.
If everything looks right, click "Install", and the setup assistant will begin installing the game.
After the game has been installed, the setup assistant will ask if you want to open the game right away, check the box if yes, and click "Finish".


You have finished installing the game!

You are now free to enjoy the game and have a good time.
